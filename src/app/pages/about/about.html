I'm Vineeth Serigar, a software engineer based in Bengaluru, India. I started my career in September 2022, right after finishing my Bachelor of Engineering in Computer Science, and I've spent the years since building production web applications used by enterprise customers.

My core identity is a frontend engineer with deep Angular expertise — TypeScript, RxJS, and NgRx are my daily tools — but I've deliberately grown into a full-stack builder. On my own time I architect and operate complete systems: custom high-performance backends on Bun and Fastify, time-series data storage on TimescaleDB, real-time WebSocket pipelines, and security-hardened authentication.

What sets my work apart is a focus on the invisible fundamentals: performance, resilience, and cost-efficiency. I like taking a hard engineering problem — decoding thousands of binary packets per second, searching a hundred-thousand-row database in milliseconds, keeping a chart perfectly continuous through a network drop — and solving it cleanly.

I also work daily with AI-assisted development tools — Claude Code, GitHub Copilot, Cursor, Codex, and Antigravity — as engineering-velocity accelerants, which lets me ship more, faster, without cutting corners on quality.
